## STEP 3: ENERGY AUDIT

```
Annual spend ÷ Unit rate = Annual kWh usage
Annual kWh ÷ 365 = Daily kWh usage
Daily kWh × Unit rate = Daily running cost
```

**Example:** £1,200 ÷ £0.21 = 5,714 kWh → 15.65 kWh/day → **£3.28/day**

---

## STEP 4: BATTERY SAVINGS

```
Daily kWh × Off-peak rate = New daily cost
Old daily cost − New daily cost = Daily saving
Daily saving × 365 = Annual battery saving
```

**Example:** 15.65 × £0.07 = £1.12/day → Save £2.16/day → **£788/year**

---

## STEP 5: SOLAR INCOME

```
Panel wattage × Number of panels = System kWp
System kWp × 4.5 peak sun hours = Daily kWh generated
Daily kWh × Export rate = Daily export income
Daily income × 365 = Annual export income
```

**Example:** 470W × 14 = 6.58 kWp → 29.61 kWh/day → £4.73/day → **£1,729/year**

---

## STEP 6: COMBINED FINANCIALS

```
Annual battery saving + Annual export income = Total annual benefit
System cost ÷ Total annual benefit = Payback period (years)
```

**Example:** £788 + £1,729 = £2,517/year → £12,567 ÷ £2,517 = **4.99 years payback**
